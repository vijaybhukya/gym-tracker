Weekly Workout Plan

Monday — Chest + Triceps

  • Barbell Bench Press — 5×5–8
  • Incline Dumbbell Press — 4×8
  • Dips (Bodyweight) — 4×8–10
  • Cable Tricep Pushdown (Straight Bar) — 3×10–12
  • Cable Chest Fly — 3×12

Tuesday — Back + Biceps

  • Barbell Deadlift — 5×5
  • Pullups / Lat Pulldown (Cable) — 4×8–10
  • Barbell Row — 4×8
  • Cable Face Pull (Rope) — 3×12
  • Barbell Curl — 3×10

Thursday — Legs

  • Barbell Squat — 5×5–8
  • Leg Press Machine — 4×10
  • Romanian Deadlift (Barbell) — 4×8
  • Hamstring Curl Machine — 3×12
  • Calf Raises (Machine/Smith) — 4×12–15

Friday — Shoulders + Traps

  • Barbell Overhead Press — 5×5–8
  • Dumbbell Lateral Raise — 4×12
  • Rear Delt Fly (DB / Machine) — 4×12
  • Dumbbell / Barbell Shrugs — 4×10
  • Hanging Leg Raises — 3×12
